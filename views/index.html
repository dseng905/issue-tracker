Issue Tracker API

Overview

The Issue Tracker is an API that keeps track issues for a project. It follows CRUD principles such that it can add, update, delete, and retrieve issues in a database. It is designed as a RESTful API using GET, POST, DELETE, and PUT requests to perform CRUD operations.

Using the route [project_url]/api/issues/[project], we can perform CRUD operations on a database powered by MongoDB. Issues will be stored and retrieved as a JSON object.
